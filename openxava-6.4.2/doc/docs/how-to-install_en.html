openxava / documentation / How to install OpenXava

First  download OpenXava from openxava.org or from SourceForge. Choose the appropriate distribution for you operating system.
Then uncompress the file you have downloaded, a zip or a tar.gz file, in any folder of your computer.
Finally, execute openxava.exe (for Windows), or openxava (for Linux and Mac). You can find this file in the root of the folder where you have uncompressed OpenXava.
For versions previous to 6.4 you don't have openxava.bat/sh, instead open the quick-start.html with your browser and follow its instructions.
If you use a 32-bit computer follow the 32-bit installation guide.
