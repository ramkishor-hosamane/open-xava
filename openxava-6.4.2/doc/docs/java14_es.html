A partir de v4.0 OpenXava necesita Java 5
Por defecto, OpenXava viene compilado con Java 5, podemos recompilarlo con Java 1.4 siguiente los siguientes pasos dentro de Eclipse:
  • Windows > Preferences > Java > Installed JREs : Escoger un Java 1.4
  • OpenXava > Properties > Java Compiler > Compiler compliance level: Escoger 1.4
  • OpenXava > Properties > Java Builde Path > Source > Quitar la carpeta java5-src
  • Build all (Ctrl -B)
Obviamente si queremos usar POJOs + JPA + anotaciones Java 5 tenemos que usar Java 5. Java 1.4 solo funciona con los componentes XML clásicos.
